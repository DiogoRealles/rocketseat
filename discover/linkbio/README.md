# HTML

**HTML:** É uma linguagem de marcação, ou seja, é responsável pela estrutura. <br />
É como os tijolos que constroem a estrutura da casa.<br />
HTML possui tags para diversas necessidades: (img, a, p, section, form, input, button, etc) <br />

HTML é dividido em duas partes: <br />
**head:** Responsável pela parte informativa e conexão com outros arquivos como: (**CSS**, **JavaScript**, **Google Fonts**, **favicon**, etc.). <br />

**body:** Responsável pela parte demonstrativa que será visível e que os usuários podem interagir. <br />
**div:** Utilizado como divisória entre os elementos. <br />
**img:** Utilizado para inserir uma imagem . <br />
**p:** Utilizado como paragrafo, conteúdo textual. <br />
**button:** Utilizado como botão, para o usuário fazer determinada ação. <br />
**span:** Utilizado como auxiliar de texto . <br />
**ul / li:** Utilizado como lista não ordenada e item desta lista respectivamente. <br />
**a:** Utilizado como ancora e/ou link entre seções e . <br />
**footer:** Utilizado para indicar rodapé tanto de uma seção como da página. <br />

# CSS

**CSS:** É uma linguagem de estilização, ou seja, é responsável pelo estilo e posicionamento dos elementos. <br />
É como a pintura e design de interior da casa. <br />

# JS

**JS:** É uma linguagem de programação, ou seja, é responsável pela interação e funcionalidades. <br />
É como as funcionalidades da casa, como: <br />
Ascender e apagar as luzes, alarmes, automação, etc. <br />
